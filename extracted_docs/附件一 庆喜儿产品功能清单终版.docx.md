# 附件一 庆喜儿产品功能清单终版.docx

庆喜儿产品功能清单

编制日期：2026年6月

1. 产品定位

庆喜儿是一款面向家庭及个体的AI情感陪伴毛绒玩具，核心价值在于 “深度记忆用户生活细节，主动提供情感关怀”。产品支持幼儿版（2-4岁）与成人版双模式：

幼儿版：侧重安全早教与习惯引导

成人版：侧重深度情感连接与跨设备互动

2. 全局功能架构

| 功能模块 | 幼儿版（2-4岁） | 成人版（≥12岁） |
| 语音交互 | 预设安全问答库 + 意图识别 | 接入DeepSeek大模型，自由对话 |
| 内容来源 | 系统内置 + 家长付费自定义 | 大模型实时生成 + 云端记忆 |
| 长期记忆 | 不记录个人数据 | 云端存储用户画像，支持变化检测 |
| 主动关心 | 仅行为引导提醒 | 情感变化检测 + 主动问候 |
| 跨设备消息 | ❌（仅单向接收家长提醒） | ✅ 情侣/亲子/闺蜜双向留言 |
| 声音克隆 | ❌ | ✅（免费1次，高级订阅不限） |
| 家长控制 | ✅ 时段锁、音量限制、内容开关、远程哄睡 | ❌ |

3. 核心功能模块详解

3.1 基础语音交互（双模式）

| 功能点 | 幼儿版 | 成人版 |
| 唤醒词 | “庆喜儿” | “庆喜儿” |
| 唤醒成功率 | 1.5m内 ≥90% | 1.5m内 ≥95% |
| 对话方式 | 预设问答库 + 有限意图识别 | DeepSeek大模型多轮对话 |
| 离线回复 | 10条预设基础回复 | 提示网络断开 |
| 内容安全 | 人工审核内容库，无暴力/色情 | 安全过滤层拦截不当输出 |
| 多轮对话 | 最多3轮，引导回主题 | 无限制 |

3.2 幼儿版专属内容库（内置 + 付费自定义）

3.2.1 系统内置内容（免费）

儿歌库：50+首，分类：中文经典、英文启蒙、行为引导、节日主题

故事库：100+个，分类：经典童话、寓言故事、原创成长、睡前故事、科普启蒙

问答库：200+条，分类：生活常识、自然科普、动物认知、数学启蒙等

3.2.2 家长付费自定义内容（订阅制）

| 自定义类型 | 功能描述 | 使用场景 |
| 自定义问答 | 家长录入“问题-答案”对（文字或录音），孩子问相应问题时优先播放家长答案 | 妈妈电话、家庭地址、安全知识 |
| 自定义提醒 | 家长创建个性化提醒内容，设定时间/周期，设备到点用家长预设语音提醒 | 吃水果、练琴、收拾玩具 |
| 自定义故事/儿歌 | 家长上传≤5分钟录音，孩子可指令播放 | 妈妈讲的故事、爸爸唱的儿歌 |
| 自定义角色音（高级） | 家长上传20秒语音，生成克隆音色用于妈妈/爸爸人格 | 妈妈不在家，孩子仍能听到妈妈声音 |

订阅模式：

基础订阅：月费9.9元 / 年费99元，含自定义问答50组、自定义提醒20条、自定义故事10条

高级订阅：月费19.9元 / 年费199元，含以上全部，且支持声音克隆2个，并开放行为数据报告

3.3 成人版专属功能

| 功能 | 功能描述 |
| 深度情感记忆 | 云端存储用户画像（作息、情绪关键词、话题偏好），检测到明显变化时主动关心 |
| 多角色人格 | 支持妈妈、爸爸、恋人、老师、闺蜜，语音/小程序切换，独立话术 + 独立TTS音色 |
| 声音克隆 | 免费版1次克隆，高级订阅不限次数 |
| 跨设备消息 | 支持情侣、亲子、闺蜜三种关系双向发送文字/语音留言，MQTT实时推送 |
| 心情日记 | 自动记录每日情绪关键词，生成周报（开心/焦虑/疲惫占比、高频词） |
| 任务系统 | 每日任务（对话5分钟、完成提醒）、双人任务（互发消息3天），获得小鱼干兑换虚拟皮肤、优惠券 |

3.4 通用功能（双模式共有）

| 功能 | 功能描述 |
| 多区域触摸感应 | 头顶、左右脸颊、胸口四区电容触摸，触发不同语音回应 + 震动反馈 |
| 仿生体感 | 胸口恒温37℃±1℃（低幼版可远程关闭）、模拟心跳震动、舵机驱动呼吸起伏 |
| 双OLED动态表情 | 双眼独立0.96寸屏，内置20+表情，根据情绪/触摸自动触发，支持OTA新增 |
| 智能提醒系统 | 支持自定义定时提醒（喝水、吃药、作息、生理期等），具备二次确认 + 三级情绪递进提醒（撒娇→闹别扭→冷战），完成后自动和解 |
| 全域OTA升级 | 固件、表情包、幼儿版内容库、语音词库均可远程升级，支持静默升级或用户确认 |
| 低功耗续航 | 连续对话≥6小时，待机（WiFi保持）≥48小时 |

3.5 家长控制中心（仅幼儿版）

| 功能 | 功能描述 |
| 使用时段控制 | 设置允许使用的时间段（如19:00-20:00），其他时间仅回复“休息时间到啦” |
| 内容分类开关 | 可关闭某一类内容（如“英文儿歌”“睡前故事”） |
| 最大音量限制 | 限制设备最大输出音量（0-100%） |
| 对话记录查看 | 查看最近20条孩子与设备的对话内容（仅文字） |
| 远程哄睡 | 一键启动哄睡模式：灯光调暗、播放指定故事、呼吸减缓 |
| 付费订阅管理 | 购买/续费、查看自定义内容使用量、管理自定义条目 |

3.6 微信小程序功能

| 页面/功能 | 功能描述 |
| 设备配网 | 蓝牙一键配网，引导式操作 |
| 设备控制台 | 查看电量、在线状态、切换模式（幼儿/成人）、切换人格、修改设备名称 |
| 提醒管理 | 增删改查自定义提醒，支持时间、周期、内容编辑 |
| 亲友圈 | 绑定好友设备，发送文字/语音消息，查看已读状态 |
| 消息中心 | 接收设备端发来的消息（如孩子发起的语音消息） |
| 任务中心 | 展示每日/双人任务，领取小鱼干奖励 |
| 个人中心 | 订单、地址、客服、订阅管理、自定义内容管理 |
| 家长控制（幼儿版） | 时段锁、内容开关、音量限制、对话记录查看、远程哄睡 |
| 商城 | 购买设备、配件、订阅服务 |
