## 7. 缺陷与修复记录

本轮参考实现自动化未发现失败用例。方案评审中发现并已在 V1.1 设计/契约中修正三项问题：

1. 原 README 建议的 `Qwen3-TTS-Flash` 不支持声音克隆，已改为 CosyVoice v3.5/v3 Flash 作为克隆音色主路由，Qwen3-TTS 仅作系统音色降级。
2. 原产品与验收文档仍使用旧唤醒词“庆喜儿”，V1.1 全部改为“阿西”，并新增模型版本、阈值、OTA 与回滚字段。
3. 原 API 缺少方言标签、置信度、来源、会话稳定性和降级原因，V1.1 已补齐。

上述属于设计修订，不是对未提供业务代码的修复。真实实现完成后，任何失败项需补充缺陷编号、根因、提交 SHA、复测日期与证据。
